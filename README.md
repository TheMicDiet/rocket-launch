# Rocket Launches
Simple Vue.js App that shows upcoming launches.
[Live version](http://launches.mi-dietrich.de)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
